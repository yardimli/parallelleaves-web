## v0.14 - 2025-09-24

### 🐛 Bug Fixes
- Fixed scroll bug when loading last position or navigating chapters

### ✨ New Features

#### Translation AI
- Add translation logging to local and remote databases
- Add logging on user edits to target text to local and remote databases
- When running translation and rephrase after the dictionary in the prompt include analysis that is not transferred to the dictionary
- Do auto analysis in the background when chapter-editor is open and save it to localstorage

#### Codex Improvements
- Simplify Codex to a single editor for the whole book
- Add temperature setting to AutoGen

#### Translation and Rephrase Enhancements
- Add tense selection to rephrase and translate editors
- Use simplified codex entries in prompts for better context
- Translate ai floating toolbar
- Add keyboard shortcuts for apply retry and cancel in floating ai results. (Esc to discard, ctrl+enter to apply, ctrl-r to retry)
- Add temperature setting

#### Editor Improvements
- Added sync scroll from target to source on markers
- Made source readonly with cursor
- Added word count display for selection in both source and target
- Added right-click context menu to add selected text to dictionary
- Add search and replace feature in chapter editor with next/previous buttons, case sensitivity, replace, replace all, and highlighting
- Add shortcut so ctrl-1 sets focus on the source, ctrl-2 sets focus on the target.
- Pressing ctrl-t brings up the translate dialog when cursor is on the target and source has selected text.
- Make ctrl-f and ctrl-h work to open search and replace bars.
- In target editor tab will go to next chapter and shift-tab will go to previous chapter

#### Chat System Enhancements
- Implemented chat history saving to localStorage
- Added ability to create new chat sessions or continue existing ones
- Added option to delete chat history from localStorage
- Added dropdown in chat history to include chapter context for chapter-specific questions
- Add temperature setting

#### Dictionary Overhaul
- Redesigned dictionary in source and target to work as classical dictionary with dedicated dialog
- Replaced textarea with row-based system allowing users to add and delete individual entries
- Add button on dictionary entries to open search replace in chapter-editor with the texts from the dictionary entry
- Add a new field for type (translation or rephrasing) if it is not set default to translation when saving the dictionary
- Increase the width of the dictionary dialog

#### Project Management
- Added "Start Blank Project" option to index.html dropdown menu alongside existing import document option
- Blank project creation requires only project name and source/target language selection
- New blank projects automatically generate structure with 3 acts and 10 empty chapters per act
- Acts use Roman numerals, chapters use regular numbers for naming

#### Chapter Editor Enhancements
- Added "Insert Chapter" function next to existing edit button for each chapter
- Added "Insert Act" and "Delete Act" functionality
- Added "Delete Chapter" functionality
- Added buttons to rename both chapters and acts for both source and target languages

#### Backup and Restore
- when exporting a book for backup also save the dictionary and new codex and restore it.
- auto backup all books every hour to a backups folder in user documents/parallelleaves/backups, keep last months backups, delete older ones.
